# Remote MCP server

This is project forked from repository, that was supplied to example in documentation for MCP auth flows. [Understanding Authorization in MCP](https://modelcontextprotocol.io/docs/tutorials/security/authorization)

Minimal implementation of a C# remote MCP server that uses Keycloak for auth.

Communication through HTTP.

# Local MCP server

This project contains local MCP server to communicate through STDIO. Based on [Build a Model Context Protocol (MCP) server in C#](https://devblogs.microsoft.com/dotnet/build-a-model-context-protocol-mcp-server-in-csharp/)
